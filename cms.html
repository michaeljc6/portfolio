Automated CMS Data Processing

This project is one of many that I have done at Direct Supply but also one of the very few I can showcase.

Direct Supply, being a healthcare company, has uses for the data collected by the CMS on Covid numbers and impacts in hospitals and Senior / Assisted living homes across the country. To do the data processing, they had been manually doing it in excel each week (about 6 hours of work). Instead, I created a program which automatically gets all of the data from CMS, does all of the relevant processing, uploads and emails the data to the relevant parties along with warnings if anything seems to be suspecious (such as large percentage jumps in cases), as well as creates a tableu powerpoint of the data. The program runs automatically on a schedule as an AWS Lambda function.

As a part of this project, I also set up a CI/CD pipeline in gitlab to automatically deploy changes to the program to the previously mentioned AWS instance.
